To install the dotfiles 

```
install git
alpine : apk add git
arch : pacman -S git / yay -S git / paru -S git
gentoo : emerge -q git

git clone https://github.com/ankledev/Alpine-dotfiles
then move over the configs you need. 
note you also need quickshell for the bar and utils.
